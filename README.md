# RPG Test Task

Консольная RPG-игра, реализованная в рамках тестового задания на позицию Automation QA Engineer.

## Описание
Игрок перемещается по подземелью, состоящему из комнат разных типов:
- St — стартовая комната
- E — комната с врагом
- Ex — выход
- ' ' — пустая комната

При наличии живого врага движение невозможно — необходимо атаковать.

## Расширения по сравнению с базовой версией

В рамках доработки тестового задания были реализованы следующие улучшения:

- Реализован полноценный автоматический пошаговый бой (AutoBattle)
  - учёт вероятности попадания оружия
  - расчёт урона с учётом брони
  - ведение лога боя

- Добавлено состояние комнаты (is_cleared)
  - комната автоматически очищается после победы над врагом
  - доступные действия игрока зависят от состояния комнаты

- Логика боя вынесена в отдельный модуль
  - контроллер не содержит бизнес-логики боя

- Реализованы автотесты:
  - проверки исходов боя
  - проверки расчёта урона
  - проверки доступных действий игрока
