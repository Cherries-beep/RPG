# RPG Test Task

Консольная RPG-игра, реализованная в рамках тестового задания на позицию Automation QA Engineer.

## Описание
Игрок перемещается по подземелью, состоящему из комнат разных типов:
- St — стартовая комната
- E — комната с врагом
- Ex — выход
- ' ' — пустая комната

При наличии живого врага движение невозможно — необходимо атаковать.

Автобой не реализован.  
При выборе действия "attack" противник погибает сразу, что является допустимым упрощением в рамках ТЗ.
